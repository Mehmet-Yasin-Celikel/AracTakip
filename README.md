# AracTakip 🚗

A Windows Forms desktop application for managing vehicle records, built with C# and Entity Framework.

## Features

- **Full CRUD** — Add, update, delete, and list vehicle records
- **Code-First Migration** — Database schema managed entirely through code
- **Auto-seeding** — Sample data loaded automatically on first run
- **DataGridView** — Click any row to populate the form fields instantly
- **LocalDB** — No external database setup required; runs out of the box

---

## Tech Stack

| Layer | Technology |
|---|---|
| Language | C# |
| UI Framework | Windows Forms (.NET) |
| ORM | Entity Framework (Code-First) |
| Database | SQL Server LocalDB |

## Getting Started

1. Clone the repository
   ```
   git clone https://github.com/Mehmet-Yasin-Celikel/AracTakip.git
   ```
2. Open `AracTakip.sln` in Visual Studio
3. Build and run — migrations and sample data are applied automatically

> Requires: Visual Studio 2022+, .NET SDK, SQL Server LocalDB (included with Visual Studio)
